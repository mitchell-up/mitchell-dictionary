# Daemon
데몬은 사용자가 직접 제어하지 않고, 백그라운드에서 돌면서 여러 작업을 하는 프로그램을 일컫는다.
- 리눅스 시스템이 처음 가동될 때 실행되는 백그라운드 프로세스이다.
- 왼도우의 서비스와 유사하하다.
- 보통 데몬을 듯하는 'd'를 이름 끝에 달고 있다.
- 일반적인 서비스는 서비스가 사용하는 포트를 관리하는 데몬이 있다.
- 데몬은 부팅 중에 메모리에 로딩되며 컴퓨터 종료시까지 유지된다.
