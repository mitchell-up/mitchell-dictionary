# Manifest
목록, 명세서를 의미하며 소프트웨어에서는 프로그램의 기본 정보와 설정이 들어 있는 파일을 의미한다.

예시로 웹에서 PWA manifest는 이렇게 JSON으로 구성한 파일이다.
```js
// web manifest (Progressive Web Apps)
{
  "name": "My PWA",
  "short_name": "PWA",
  "start_url": "/",
  "display": "standalone",
  "theme_color": "#ffffff"
}
```
